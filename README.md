## Getting Started

### Prerequisites
- [Git](https://git-scm.com/)
- [Node.js](https://nodejs.org/) (LTS recommended; needed once React tooling is added)

### Setup
1. Clone the repository:
   ```bash
   git clone https://github.com/<your-username>/frontend-ai-capstone.git
   cd frontend-ai-capstone
   ```
2. Open the project in your editor of choice.
3. As the frontend is built out, install dependencies and start the dev server using the commands documented in this section.

> **Note:** The project is in early setup. React and Node.js tooling will be added in upcoming milestones.
